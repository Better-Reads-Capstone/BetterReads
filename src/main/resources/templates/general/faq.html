Frequently Asked Questions

    What technologies power the site?
  • Java spring boot, MySQL, JavaScript, jQuery, and the Bootstrap framework cover the main technology stack. There are various helper libraries in use for additional feature functionality. Check out the GitHub link in the footer for full details on the code.
    What was the inspiration for the site?
  • The Goodreads site was a major influencer of our project. The functionality of that site is massive, but we felt the user experience could use a more modern approach so our aim at BetterReads was to implement similar functionality while giving the user experience an overhaul.
